About CSS

CSS (Cascading Style Sheets) is used to style and design web pages. It controls colors, fonts, layouts, and spacing. With CSS, plain HTML becomes attractive and user-friendly. For example, you can change the background color of a page, center headings, or add hover effects to buttons without using JavaScript.

Hover me

Basic CSS Example

This short CSS example sets a light blue background for the page and makes the heading text blue:

<style>
  body { background: lightblue; }
  h1 { color: blue; }
</style>

More CSS Examples

1) Changing Text Color & Font
