About HTML

HTML (HyperText Markup Language) is the standard language for structuring content on the web. It uses tags to define headings, paragraphs, links, images, lists, and more.

Semantic tags like <header>, <nav>, <article>, and <footer> help with accessibility and SEO by giving meaning to content.

An HTML page typically includes:

  • A DOCTYPE declaration
  • The root <html> element
  • A <head> section with metadata like the title and character set
  • A <body> section with visible content
